# Security Policy

This repository contains documentation and templates. It does not ship runtime
software or hosted services.

## Reporting

If you find a security-relevant issue in the templates, such as advice that
encourages unsafe credential handling, destructive automation, or publication of
private data, please open an issue or contact the maintainer through the
available GitHub profile channels.

Do not include secrets, private logs, access tokens, or confidential project
material in public issues.

## Scope

In scope:

- unsafe workflow guidance;
- templates that encourage destructive actions without review;
- examples that could leak private data if followed literally.

Out of scope:

- vulnerabilities in third-party agent products;
- issues caused by local modifications outside this repository;
- requests for security support unrelated to these templates.
