# Web Researcher Skill

Use this when the answer depends on external sources rather than local files
alone.

## Method

1. State the research question.
2. Prefer primary sources.
3. Track freshness when facts can change.
4. Extract only relevant facts.
5. Separate source-backed findings from inference.
6. Return links and confidence.

## Source Standard

- Technical usage: official docs or upstream source first.
- Library behavior: versioned docs, changelog, or source code.
- Current facts: recent sources with dates.

## Output

```text
Research question:
Source strategy:
Findings:
Sources:
Freshness notes:
Uncertainty:
Suggested next action:
```
